# React Hooks Challenge

A collection of 8 mini-projects demonstrating every React hook, built with **Vite + React 19 + TypeScript** and zero external libraries.

## Projects

| Project | Hooks |
|---------|-------|
| [Kanban Board](./kanban.html) | useState, useReducer, useRef, useCallback, useMemo, useLayoutEffect, useImperativeHandle |
| [Real-time Chat](./chat.html) | useState, useEffect, useRef, useOptimistic, useCallback, useTransition, useSyncExternalStore |
| [Form Wizard](./form-wizard.html) | useState, useEffect, useRef, useActionState, useTransition, useId, useCallback |
| [E-commerce](./ecommerce.html) | useState, useContext, useReducer, useMemo, useCallback, useTransition, useDeferredValue |
| [Dashboard](./dashboard.html) | useState, useEffect, useRef, useMemo, useCallback, useTransition, useDebugValue, useSyncExternalStore |
| [Markdown Notes](./notes.html) | useState, useEffect, useRef, useInsertionEffect, useCallback, useMemo, useImperativeHandle, useDeferredValue |
| [Event Calendar](./calendar.html) | useState, useReducer, useRef, useLayoutEffect, useMemo, useCallback, useDebugValue |
| [Poll / Voting](./poll.html) | useState, useEffect, useContext, useReducer, useCallback, useMemo, useOptimistic, use, useTransition |
